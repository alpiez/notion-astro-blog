# Notion Astro Blog

A blog built with Astro that fetches pages from your Notion.

## How to setup

Please follow this guide: [HOWTOSETUP.md](HOWTOSETUP.md)

## Run the server

```bash
# For development
npm run dev

# To build and deploy
npm run build
npm run preview
```

## Supported blocks

- ✅ paragraph
- ✅ heading 1, 2, 3
- ✅ bulleted list
- ✅ numbered list
- ✅ toggle
- ✅ quote
- ✅ callout
- ✅ code
- ✅ image
